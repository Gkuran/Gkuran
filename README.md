## 📚 BGSR - Biodiversity and Geoscience Spatial Registry
[BGSR](https://bgsr.gabrielkuran42.workers.dev/?scenario=default) is an open-source, map-first scientific platform for exploring biodiversity and geoscience in the same spatial context. It brings together fauna, flora, soil, and geological information in a single interface built for regional analysis, field-oriented research, and environmental interpretation.
